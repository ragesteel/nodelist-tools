# Инструменты для работы с нодлистами, для FIDONet Technology Networks
Пока только обработка Diff'ов и проверка CRC. 

Хочется сделать инструмент у которого по сути два параметра:
путь к исходному каталогу с нодлистами и набор путей к диффам.
Разумеется вся распаковка должна быть прозрачной, если речь про ZIP-архивы.

Дальше алгоритм простой — проходим по всем нодлистам в каталоге, нодлистом считаем файлы с расширением из трёх цифр, и сохраняем сопоставление «заголовок» → «имя файла».
Потом идёт по Diff'ам и ищем нужный заголовок.

Если находим → применяем этот Diff. И ещё один проход по Diff'ам.
